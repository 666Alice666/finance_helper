# Finance Helper
Finance Helper — веб-приложение для управления персональными финансами. Система позволяет вести учет доходов и расходов, управлять категориями, просматривать историю операций, анализировать статистику и выполнять базовые расчеты по кредитам и займам.
Проект разработан на Python с использованием Flask и PostgreSQL.
# Возможности
* Регистрация и авторизация пользователей.
* Ролевая модель доступа: `user` и `admin`.
* Защита пользовательских данных: пользователь работает только со своими категориями и транзакциями.
* Создание, просмотр, обновление и удаление категорий доходов и расходов.
* Создание, просмотр, обновление и удаление финансовых транзакций.
* Просмотр истории операций.
* Просмотр статистики по доходам, расходам, балансу и категориям.
* Калькулятор кредитов и займов.
* REST API с JSON-ответами и префиксом `/api`.
* CSRF-защита HTML-форм и изменяющих запросов.
* Фаззинг-тестирование REST API.
